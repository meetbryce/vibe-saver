# Vibe Saver Website

This directory contains the website for the Vibe Saver project, hosted online.

## Development

The website is built with plain HTML, CSS, and JavaScript.

To run the website locally, you can use any simple HTTP server, for example:

```bash
# Using Python
python -m http.server 8000

# Or using Node.js
npx serve
```

Then open your browser to http://localhost:8000

## Deployment

The website is automatically deployed when changes are pushed to the main branch.
